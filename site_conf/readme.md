# ⚠️ DEV NOTICE: Preset Management System

This section is the "Brain" of the project. 

### Why do we have this?
To avoid hardcoding values like colors, site titles, or feature-flags. 

### How to use:
1. Create a new Configuration object.
2. Fill in the parameters (colors, names, etc.).
3. Check the `is_active` box and save.
4. The system will automatically switch the entire site to this preset.
